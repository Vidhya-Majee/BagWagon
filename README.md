👜 BagWagon
BagWagon is a sleek and modern e-commerce web application designed for bag enthusiasts. Built using the MERN stack with a twist—EJS for templating and Tailwind CSS for styling—it offers a seamless shopping experience with a clean, responsive UI and secure backend.

🚀 Tech Stack
EJS – Dynamic server-side rendering

Express.js – Fast and minimalist backend framework

MongoDB + Mongoose – Robust NoSQL database and elegant ODM

Tailwind CSS – Utility-first CSS for rapid UI development

🛍️ Features
🧭 Browse a curated collection of bags

🛒 Add items to your cart

💼 Purchase securely with a streamlined checkout

📱 Fully responsive design for mobile and desktop

🔐 Secure user authentication and data handling

🧪 Future Enhancements
User reviews and ratings

Payment gateway integration

Admin dashboard for inventory management
